#### Python: Async

In this directory, I learn about handling asynchronous coroutines and tasks with Python.

Some of the concepts I learn in this project are:
- Coroutines
- Tasks
- The `asyncio` module
- Breaking down the Asyncio module:
    - Event Loop
    - Coroutine
    - Tasks
    - Future
- Creating and Running Coroutines using:
    - `async def`
    - `await`
    - `asyncio.run()`
- `asyncio` Methods:
    - `sleep()`
    - `gather()`
    - `get_event_loop()`
    - `run()`
    - `new_event_loop()`
    - `run_forever()`
    - `run_until_complete()`
    - `stop()`
    - `create_task()`
    - `stop()`
    - `close()`
    - `get_loop()`
- Certain Async Statements and their Uses:
    - `async def ...`
    - `await`
    - `async with`
    - `async for`
    - `async`
- The difference between a `Coroutine` and a `Task`
